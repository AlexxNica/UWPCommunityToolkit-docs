UWP Toolkit

The UWP Community Toolkit provides easily configurable controls, code, syntax, APIs and more, that allow developers to build, deploy, and launch functional and visually appealing applications for Windows.

UWP, combined with Windows 10, makes it easier to develop apps with just one API set, one app package, and one store to, build, deploy and launch apps on any Windows platform. This toolkit offers installation instructions, open-source code and samples apps to build your own sample app project and configure services such as Bing, Twitter, and Facebook.

To access Introduction, Installation Steps, Controls and more, refer to the UWP Community Toolkit Getting Started Guide.

UWP Features

Animations

Behaviors features allow you to implement specific behaviors and apply a specific composition to your application, such as Blur and Opacity.

Controls

Controls allow you to build robust architecture for Windows apps. They include both layout and foundation types, including HamburgerMenu, RadialGauge and more.

Helpers

Helpers enhance your application formats. Windows helpers in this toolkit include Color, Connections, StorageFiles, and Streams.

Notifications

Notifications help design tiles and interactive toasts for developing Windows apps. They include LiveTiles, Toasters, and WeatherLiveTileandToast.

Services

Services lets you retrieve or publish data to your Bing, Facebook or Twitter service. By adding a Windows store ID, you can easily build Windows apps for these services.
