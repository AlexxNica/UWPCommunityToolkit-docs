UWP Community Toolkit

This toolkit can be used by any new or existing UWP application written in C# or VB.NET. Our goal is to support the latest and previous stable release of the SDK. Today the toolkit is compatible with apps developed with Windows 10 SDK Build 10586 or above.

Build UWP apps for any Windows 10 device, including PC, Mobile, XBOX, IoT and HoloLens. You can also use the toolkit with an existing desktop app converted to UWP using the Desktop Bridge.

We have published the UWP Community Toolkit Sample App to the Windows Store. Download the app and play with the controls live on your own devices.

Read the Getting Started page for instructions on how to use our NuGet packages in your new or existing applications.

UWP Community Toolkit Features

Animations

Animations allow you to implement specific XAML behaviors and apply visual composition to your application, such as Blur and Fade. You can also use code to chain animations together without using XAML.

Controls

Controls were built to be fast and up to current UWP coding standard. They include HamburgerMenu, RadialGauge, SlideableListItem and more.

Code Helpers

Code helpers are classes that help you build modern applications quickly. These include Color conversion, Internet connection detection, StorageFile helpers, and more.

Notifications

Notifications are a key UI component to help UWP applications with user engagement. Live Tiles and Toast samples help you get started using them fast.

Services

Modern applications need to connect to cloud services for social and search capabilities. Implementing them using this toolkit is as simple as possible. Bing, Facebook, and Twitter services are supported.
